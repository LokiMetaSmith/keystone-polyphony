# Technology Stack\n\n## Languages\n- **Python 3:** Primary language for server-side logic, agent orchestration, and coordination primitives.\n- **JavaScript (ES6+):** Used for the Node.js P2P sidecar and the browser-resident dashboard.\n- **HTML5 / CSS3:** Standard web technologies for the user interface.\n\n## Frameworks & Libraries\n### Backend & Coordination\n- **aiohttp:** Asynchronous HTTP server for the Liminal Bridge.\n- **pydantic:** Data validation and settings management using Python type annotations.\n- **Hyperswarm:** A distributed networking stack for peer discovery and connectivity (running in a Node.js sidecar).\n- **Model Context Protocol (MCP):** For standardized interaction between agents and tools.\n\n### Frontend\n- **React 18:** Component-based UI library for the Interactive Dashboard.\n- **Vite:** Next-generation frontend tooling for fast development and building.\n- **react-force-graph-2d:** Visualization of the distributed network mesh.\n\n### Security & Identity\n- **cryptography (Python):** For server-side cryptographic operations and secure communication.\n- **TweetNaCl (JS):** Cryptographic library for browser-side Ed25519 identity management.\n\n## LLM Integrations\n- **OpenAI / Anthropic / Google Gemini:** Supported cloud-based model providers.\n- **Ollama:** Support for local model execution.\n\n## Development & Testing\n- **pytest & pytest-asyncio:** Testing frameworks for verifying core logic and asynchronous operations.\n- **flake8 & black:** Linting and code formatting for Python.\n- **Vite Preview:** For local verification of dashboard builds.
