# Initial Concept\nKeystone Polyphony: A Mind Bridge for collaborative multi-agent and human coordination using RTOS-inspired primitives.\n\n# Product Overview\nKeystone Polyphony is a distributed ecosystem designed to facilitate seamless collaboration between humans and autonomous AI agents. By abstracting complex coordination challenges into Real-Time Operating System (RTOS) primitives—such as Liminal Spaces (shared context), Batons (resource locks), and Signals (event notifications)—it provides a robust framework for decentralized work.\n\n# Core Mission\nTo build a "Mind Bridge" where diverse contributors can work in tandem with full agency, ensuring that project state is coherent and dependencies are managed without a central monolithic authority.\n\n# Target Audience\n- **Architects & Engineers:** Building the infrastructure for multi-agent orchestration.\n- **Conductors & Strategists:** Organizing workflows and managing project dependencies.\n- **Autonomous Agents:** AI systems performing tasks, triaging issues, and submitting code.\n- **Creative Contributors:** Storytellers and designers shaping the project's identity and UX.\n\n# Key Features\n- **Liminal Bridge:** A peer-to-peer mesh network for synchronizing state and thoughts.\n- **Interactive Dashboard:** A real-time UI for visualizing the network, managing identity, and acquiring resource batons.\n- **RTOS Coordination Primitives:** Cryptographically secured mutexes (Batons) and event groups (Signals).\n- **Identity-Based Authentication:** Passwordless registration using browser-resident Ed25519 key pairs.\n- **Multi-Provider LLM Integration:** Built-in support for OpenAI, Google Gemini, Anthropic, and local Ollama models.\n\n# Success Criteria\n- Reliable, low-latency synchronization of global project state across a distributed mesh.\n- Zero-touch activation of new worker nodes via standardized bootstrapping scripts.\n- Intuitive visualization of agent "thoughts" and project progression.\n- High degree of autonomy for agents to self-select and execute tasks safely.
